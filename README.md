# Negativity Sells: A Sentiment Analysis of Huff-Post Political Articles


This repository contains an R Markdown and a dataset for sentiment analysis of Huff-Post headlines, along with a corresponding paper to discuss conclusions.

# Dataset

* Link to dataset used: https://www.kaggle.com/datasets/rmisra/news-category-dataset/data

#Repository Structure

* Sentiment_Analysis.Rmd - R Markdown with all code.
* Sentiment_Analysis.docx.pdf -  Associated paper discussing the project motivation, process, and results.

# What the Markdown Does

* Loads and cleans the data.
* Performs Bing sentiment analysis and NRC emotional analysis.
* Conducts linear regression tests to determine whether Political article headlines are more negative than lifestyle news articles.
* Conducts linear regression tests to determine whether headlines are more negative than actual news articles.
* Creates Pie-Charts, Bar Graphs, a Box and Whisker Plot, and Word Clouds to visualize the data.
